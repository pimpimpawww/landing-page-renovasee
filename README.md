# Renovasee - Design & Build Contractor Landing Page

Landing page modern untuk perusahaan konstruksi Renovasee yang dibangun dengan Next.js 14, Tailwind CSS, dan TypeScript.

## Fitur

- ✨ Design modern & clean
- 📱 Fully responsive & mobile-first
- 🎨 Tailwind CSS untuk styling
- ⚡ Next.js 14 dengan App Router
- 🎭 Smooth animations & transitions
- 📊 Counter animation pada statistik
- 🧮 Kalkulator biaya konstruksi
- 📸 Portfolio gallery dengan hover effects
- 🔝 Scroll to top button
- 🎯 SEO optimized

## Struktur Komponen

```
components/
├── Navbar.tsx       # Sticky navbar dengan blur effect
├── Hero.tsx         # Hero section dengan parallax
├── Features.tsx     # 3 card highlight features
├── Calculator.tsx   # Form kalkulator biaya
├── Stats.tsx        # Statistik dengan counter animation
├── About.tsx        # Tentang perusahaan
├── Portfolio.tsx    # Gallery portfolio
├── CTA.tsx          # Call-to-action banner
└── Footer.tsx       # Footer dengan scroll to top
```

## Instalasi

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser di [http://localhost:3000](http://localhost:3000)

## Build untuk Production

```bash
npm run build
npm start
```
